# Release Notes

## 0.1.0

Initial preview release of WoadStoat.MarkovNames.

### Added

- Character-based Markov name generation.
- Token-based Markov name generation.
- Culture and category model libraries.
- JSON culture profile loading from files and strings.
- Data-file culture profiles.
- Guided prefix generation.
- Guided suffix generation.
- Configurable suffix join modes.
- Generation filters and custom validators.
- Deterministic seeded generation.
- Character model serialisation.
- Token model serialisation.
- Unified high-level `NameGeneratorLibrary` API.
- Unit tests.
- NuGet package metadata.
- GitHub Actions build, test, and pack workflow.

### Notes

This is an early preview release. Public APIs may still change before 1.0.0.
